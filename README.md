# Chat App

A real-time one-to-one chat application built using:

- React.js
- Node.js
- Express.js
- MySQL
- Socket.io

## Features

- User Registration
- User Login
- One-to-One Chat
- Real-time Messaging
- Message Storage in MySQL
- Logout Functionality

## Installation

Frontend:
npm install
npm start

Backend:
npm install
npm start
